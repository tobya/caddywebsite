http.requestid

requestid when invoked generates a UUID which can be referenced via the `{request_id}` placeholder , for use in many other directives including `header` and `log`

When requestid directive is present `{request_id}` will have a value otherwise it will be empty

Syntax

requestid simply has to be included on a line on its own for the requestid to be generated.

requestid

Examples

Send a requestid header

requestid
header / "X-Request-Id" {request_id}
